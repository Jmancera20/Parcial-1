PROYECTO DE CÁLCULO DE PAGO DE ÓRDENES DE COMPRA

Este proyecto tiene como objetivo desarrollar un sistema que calcule la forma de pago de las órdenes de compra de una empresa cliente. El cálculo se hace con base en el monto total de compra y se determina cuánto se paga con inversión propia, cuánto se pide prestado al banco y cuánto se paga al fabricante con intereses.
